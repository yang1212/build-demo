## 🚀 操作步骤

### 1、如存在旧版本：卸载掉旧版本  

```
npm uninstall vue-cli -g 
```
### 2、安装@vue/cli
对于 Vue 3.x 的项目，需要使用 Vue CLI v4.5 以上的版本
```
npm install @vue/cli@5.0.8 -g
```

### 3、创建项目
```
vue create vue3_demo
yarn install
```

### 4、引入loader
```
"sass": "^1.45.0",
"sass-loader": "12.4.0"
```
