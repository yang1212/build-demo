## v5_vue3

Vue-cli4 需要 Node.js v8.9 或更高版本 (推荐 v10 以上)
## 版本
当前运行版本:
```
node: v12.0.0
```

<br/>

## 🚀 操作步骤

### 1、如存在旧版本：卸载掉旧版本  

```
npm uninstall vue-cli -g 
```
### 2、安装@vue/cli
对于 Vue 3.x 的项目，需要使用 Vue CLI v4.5 以上的版本
```
npm install @vue/cli@4.5.0 -g
```

### 3、创建项目
```
vue create v5_vue3
```
